## Checklist

1. Is Elixir still the control plane?
2. Are Zig and external runtimes still behind narrow boundaries?
3. Is there a deterministic fallback if native, provider, or subprocess execution fails?
4. Does the change preserve `.claw/sessions` replayability?
5. Is any new shell or write capability explicitly gated?
6. Was at least one focused automated test added or strengthened?
7. Was at least one smoke command run for the changed operator surface?
8. Did the diff stay small relative to the behavior added?
